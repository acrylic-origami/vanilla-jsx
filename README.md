Vanilla-JSX
=========

Vanilla-JSX is a library that converts JSX into vanilla HTML Elements.

#### Example
```
'use babel'
/** @jsx vanilla.jsx */
import vanilla from 'vanilla-jsx'

console.log(
  <div class="Wow">Something <span>Special</span></div>
)
```
